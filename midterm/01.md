### gcc用法

```
gcc hello.c -o hello       # 創建hello.exe
gcc -c fib.c -o fib        # 只編譯不連結
gcc -S fib.c -o fib.s      # 編譯成組合語言
gcc main.c sum.c -o run    # 自訂library且只編譯不連結
```

### git分支

```
git branch                 # 查看分支
git checkout -b master     # 建立新分支
git checkout main          # 切換分支
```
